Background

I have used Python for training a CNN model and then C# to build a client application. A basic knowlege of C#, NUGET and Python would be very helpful. I have used Visual Studio 2017 for the simple Console application and Visual Studio Code for Python(1.36.1) training scripts. In this article I have used the GPU version of the Tensorflow for to improve the speed of learning. You would need a GPU enabled desktop. The reader is cautioned that coding with the GPU requires additional CUDA libraries and drivers to be installed. This article also assumes that the reader is familiar with the basic principles of Deep Convolution Neural Networks.
